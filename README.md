# Chaos Mesh (chaos-mesh)
Chaos Mesh is a cloud-native chaos engineering platform that orchestrates chaos experiments on Kubernetes environments to test system resilience and reliability.

## APIs

### Chaos Mesh API
Chaos Mesh provides Kubernetes custom resources and a REST API for orchestrating chaos experiments including network faults, pod failures, IO chaos, stress testing, and HTTP request injection. The Chaos Dashboard provides a web UI backed by a REST API for creating, managing, and monitoring chaos experiments and workflows.
